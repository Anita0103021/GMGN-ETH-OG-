# GMGN-ETH-OG检测
检测 GMGN 的 OG 系列工具

## 📌 项目简介
本项目是一个用于检测 GMGN 平台 OG 系列相关内容的工具，包含扫描扩展程序，可帮助用户快速识别相关信息。

## 📦 文件说明
- `gmgn-scanner-ext_19.zip`：GMGN 扫描扩展安装包，可用于浏览器扩展安装。

## 🚀 使用方法
1.  下载 `gmgn-scanner-ext_19.zip` 文件
2.  将压缩包解压到本地文件夹
3.  在浏览器中开启「开发者模式」，加载已解压的扩展程序
4.  安装完成后即可使用相关检测功能
